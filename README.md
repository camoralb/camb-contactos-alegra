# camb-contactos-alegra

![](https://camb.com.ve/camb-contactos/capture.jpg)

###Caracteristicas del desarrollo

- PHP Zend Framework 1.12.
- ExtJs 4.2.2.
- Famfamfam Silky icons.



####Implementación

1. Clonar este proyecto via git clone en su servidor web. Es importante configurar un virtualhost. Para el caso de apache2 puede ver un ejemplo del archivo de configuración en el directorio docs/README.txt.

2. Configurar en application/configs/application.ini el URI (Dirección URL del API de Alegra), el email y el token con el cual se debe conectar el cliente al API.

3. Puede ver un demo de la implementación [aquí](http://contactos-alegra.camb.com.ve/ "aquí").
